- 오늘은 뭘 먹을지 정해주는 프로그램.

## 사용된 기술
- Java
- Spring boot
- Gradle
- h2 database
- jpa

## 구현된 API
- 사용자 등록
- 메뉴 조회
- 랜덤으로 메뉴 선정
- 오늘 먹은 메뉴 저장
- 최근 먹은 메뉴 조회
- 싫어하는 메뉴 저장

### 사용자 등록 API
- 닉네임을 받아 사용자를 등록한다.

### 메뉴 조회
- 모든 메뉴를 조회 한다.
- 싫어하는 메뉴가 있을 시 그 메뉴는 제외한다.

### 랜덤으로 메뉴 선정
- 모든 메뉴 중에 랜덤으로 하나의 메뉴를 반환한다.
